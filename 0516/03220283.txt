課題２のC++のクロシージャで、値のキャプチャの書き方が気になったので調べてみた

構文は
[キャプチャリスト](パラメータリスト) mutable 例外仕様 属性 -> 戻り値の型 { 関数の本体 }
のようになっている。（cppreferenceより）

[](auto x){ return x; } // キャプチャしない
[y](auto x){ return x + y; } // yをコピーして渡す
[&y](auto x){ return x + y; } // yを参照で渡す

[=](auto x){ return x + y; } // すべてコピー渡し 
[&](auto x){ return x + y; } // すべて参照渡し

&が参照渡しなのは関数の引数を取るときに馴染みがあるのでなるほどと思った。

コピーが行われるのは関数を定義した時で、下のようになるのは面白いが思わぬバグを引きそうなので大変だと思った。

auto x = 1;
y = [x](){return x;}
x++; // ここで増やしても、
int z = y(); // z = 1
